# 接口定义

## 我的收藏
```
接口 GET方式 /collection/conllectionlist/list.json
```

入参
```
userID Integer 用户ID
pageNo Integer 当前页数
pageSize Integer 分页数
```

出参
```
flag        Integer     业务判断标志位，0代表成功，-1代表失败
msg         String      提示信息
pageNo      Integer     第几页
pageCount   Integer     总页数
recordCount Integer     记录总数
pageSize    Integer     一页显示多少条
items       List        收藏夹列表
````

```
list：
    productName                 String      商品名称
    productDes                  String      商品介绍
    productActivity             String      商品参加的活动名称
    productHref                 String      商品链接地址
    productImg                  String      商品图片地址
    productOriginalPrice        demail      商品原价
    productPreferentialPrice    decimal     商品优惠价
```
## 我的地址

### 添加地址
```
接口 POST方式 /address/addressoperation/addaddress.json
```

入参
```
userID              Integer     用户ID
receiverName        String      收货人名称
receiverPhone       String      联系电话
receiverProvince    String      所在省
receiverCity        String      所在城市
receiverRegion      String      所在区域
receiverRemark      String      街道/小区
defaultAddress      Integer     1代表默认，0表示不设为默认地址
```

出参:
```
flag        Integer     业务判断标志位，0代表成功，-1代表失败
msg         String      提示信息
```

### 编辑地址
```
   接口 POST方式 /address/addressoperation/updateaddress.json
   ```

   入参
   ```
   userID              Integer     用户ID
   addressSyscode      String      地址syscode(或者Id，根据底层接口决定，暂定为syscode)
   receiverName        String      收货人名称
   receiverPhone       String      联系电话
   receiverProvince    String      所在省
   receiverCity        String      所在城市
   receiverRegion      String      所在区域
   receiverRemark      String      街道/小区
   defaultAddress      Integer     1代表默认，0表示不设为默认地址
   ```

   出参:
   ```
   flag        Integer     业务判断标志位，0代表成功，-1代表失败
   msg         String      提示信息
   ```

### 删除地址
```
接口 POST方式 /address/addressoperation/deladdress.json
```

入参
```
userID              Integer     用户ID
addressSyscode      String      地址syscode(或者Id，根据底层接口决定，暂定为syscode)
```

出参:
```
flag        Integer     业务判断标志位，0代表成功，-1代表失败
msg         String      提示信息
```
